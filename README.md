Bike道
## サイト概要
ロードバイクで走った場所やそこにあったカフェなどをタグ付け、投稿するSNSサイト

### サイトテーマ
オリジナルを全国に

### テーマを選んだ理由
どこにいくか考えることが1つの楽しみであり、その自身でつくったオリジナルルートやその場所ならではのおしゃれなカフェなどを
全国のロードバイク乗りの方に共有し合って欲しい


### ターゲットユーザ
ロードバイクに乗っている方

### 主な利用シーン
ロードバイクに乗り始める時、目的地に到着した際
